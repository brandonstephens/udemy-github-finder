# Udemy Github Finder

<https://www.udemy.com/course/modern-react-front-to-back/>

## Setup

Make an app on https://github.com/settings/applications

1. `touch .env.local`
2. add client id and secret to `.env.local`

3. `nvm use`
4. `npm install`
5. `npm start`
